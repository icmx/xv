# xv

[xkcd.com](https://xkcd.com/) viwer app, done in pure JavaScript and Webpack.

## Development

```sh
# clone repository
git clone https://github.com/icmx/xv

# go to local copy
cd xv

# install dependencies
npm install

# launch it on localhost:8000
npm run serve

# build it
npm run build
```
